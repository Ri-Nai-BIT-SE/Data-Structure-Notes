# 计算理论

计算理论是计算机科学的理论基础，研究计算的本质、能力和限制。本部分包含以下章节：

## [计算模型](计算模型/)
有限自动机、正则语言、图灵机等基本计算模型的定义与性质。

## [可计算性](可计算性/)
可计算性理论基础，包括递归函数、不可判定问题等内容。

## [计算复杂性](计算复杂性/)
时间复杂性和空间复杂性分析，P类和NP类问题的讨论。
